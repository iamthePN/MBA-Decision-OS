# User Guide

This guide explains how students and admins use MBA Decision OS after setup.

## Student Workflow

### 1. Sign Up Or Log In

- New users open `/register`
- Existing users open `/login`

### 2. Build The Profile

Open `Profile Builder` and fill in:

- academics
- entrance exam score
- work experience
- achievements
- specialization preferences
- location preference
- budget range
- risk appetite
- short-term and long-term goals
- preferred industries
- funding mode
- scholarship need
- target intake year

### 3. Review Profile Evaluation

Open `Evaluation` to see:

- profile strength score
- readiness index
- score contribution cards
- explanation of current strengths and gaps

### 4. Discover Colleges

Open `Discovery` to:

- search colleges
- filter by location, fees, exam, and ROI
- sort by recommendation, fees, salary, ROI, or admission probability
- open detail pages
- shortlist programs

### 5. Read College Detail Pages

Each detail page includes:

- fees and salary stats
- profile match
- ROI score
- payback period
- placement sectors
- recruiter data
- risk fit
- recommendation explanation

### 6. Review Recommendations

Open `Recommendations` to see a ranked list of personalized colleges with plain-English reasons.

### 7. Compare Colleges

Open `Compare` and place up to three colleges side by side.

### 8. Use The Digital Twin Simulator

Open `Digital Twin` and choose:

- a college
- specialization
- risk appetite
- funding mode
- expected role path

The simulator will project:

- total investment
- placement probability
- expected CTC range
- 3-year and 5-year earnings
- payback period
- role likelihood
- safe / balanced / high-risk scenario values

### 9. Track Applications

Open `Applications` to:

- add a college
- set status
- capture deadlines
- save notes
- record fees paid
- define next steps

### 10. Export Results

Open `Settings` to:

- export shortlist CSV
- export application tracker CSV
- open the printable recommendation report
- update account details
- change password
- toggle theme

## Admin Workflow

### 1. Log In As Admin

Use the seeded admin credentials from the README.

### 2. Manage Colleges

Open `Admin > Colleges` to:

- create a new college
- edit an existing college
- delete a college
- import colleges from CSV
- toggle featured colleges

### 3. Tune Scoring

Open `Admin > Scoring` to edit weights for:

- profile scoring
- match scoring
- ROI scoring
- final ranking

### 4. Monitor Users And Pipeline

Open `Admin > Users` to inspect:

- all users
- shortlist counts
- application counts
- current application statuses

### 5. Manage Content And Catalogs

Open `Admin > Content` to:

- add testimonials
- add pricing plans
- view contact messages
- add exams, recruiters, and sectors

## Tips

- Refresh recommendations after meaningful profile edits.
- Keep scoring weights balanced within each group.
- Use the compare page before committing to a high-fee program.
- Treat the simulator as a decision-support tool, not a guarantee engine.
